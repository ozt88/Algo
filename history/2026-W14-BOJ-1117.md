# BOJ 1117 색칠 1

## 문제 요약
- W×H 종이를 세로(x=f)로 1회 접고 가로로 c회 더 접은 뒤 [x1,x2]×[y1,y2] 직사각형을 칠함
- 종이를 다시 펼쳤을 때 색칠되지 않은 면적을 구하는 문제
- W, H ≤ 10^9, 0 ≤ f ≤ W, 0 ≤ c ≤ 10^9, 0 ≤ x1 < x2 ≤ f, 0 ≤ y1 < y2 ≤ H/(c+1)
- 출력: 색칠되지 않은 면적 (int64 필수)

## 핵심 알고리즘
- **독립성**: x방향(세로 접기)과 y방향(가로 접기)은 서로 독립 → 색칠 면적 = painted_x × painted_y
- **x방향**: 겹치는 구간 overlap = min(f, W-f). [x1,x2]에서 overlap 이하 부분은 펼치면 2배.
  - doubled = max(0, min(x2, overlap) - x1), single = (x2-x1) - doubled
  - painted_x = doubled×2 + single
- **y방향**: 가로 c회 접기 → c+1개 구간 동일하게 칠해짐
  - painted_y = (y2-y1) × (c+1)
- **답**: W×H - painted_x × painted_y

## 실패/버린 아이디어
- 직접 좌표 펼치기 시뮬레이션: 접힌 각 점을 원래 좌표로 역매핑해 면적합산 시도
  → O(N) 루프가 필요해 제한 시간 초과 우려로 기각
- 직접 수학적 분석으로 x/y 독립성을 확인하여 O(1) 풀이 도달

## 최종 선택 이유
- x방향과 y방향이 완전히 독립이므로 각각 1D 문제로 환원 후 곱셈 → O(1) 풀이
- 겹치는 구간(overlap)을 명시적으로 계산하면 doubled/single 분류가 단순
- int64 곱셈으로 W×H ≤ 10^18 범위를 오버플로 없이 처리
