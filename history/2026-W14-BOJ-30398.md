# BOJ 30398 대구일과학고등학교

## 문제 요약

- d차원 격자에서 (1,...,1)→(N,...,N)으로 P개 중간점을 순서대로 거치는 경로 수를 mod 10^9+7로 출력
- 경로: 각 축 방향으로만 이동 (좌표 증가 방향), 역방향 이동 불가
- 제약: 1 ≤ N ≤ 1000, 1 ≤ d ≤ 1000, 0 ≤ P ≤ 1000, N×d ≤ 10^6
- 출력: 경로 수 mod 10^9+7 (불가능한 경로는 0)

## 핵심 알고리즘

- **D-01 구간 분할**: 시작점→점1→점2→...→점P→끝점 = P+1개 구간으로 분리, 각 구간 독립 계산
- **D-02 다항계수**: 각 구간의 경로 수 = 다항계수(다중집합 순열). 차원별 이동 수가 고정되므로 이동 순서만 결정
- **D-03 공식**: 다항계수 = S! / (d_1! × d_2! × ... × d_D!), S = Σ(B_j - A_j) (택시 거리)
- **D-04 최종 답**: P+1개 구간 경로 수의 곱 (mod 10^9+7)
- **D-05 전처리**: 팩토리얼 + 역팩토리얼을 0~N×d까지 전처리 (O(N×d)), N×d ≤ 10^6이므로 충분
- **D-06 역팩토리얼**: 페르마 소정리 — inv(k!) = (k!)^(MOD-2) mod MOD, 이후 역방향 재귀로 전체 계산
- **D-07 불가능 처리**: 임의 구간에서 B_j - A_j < 0이면 즉시 0 반환 (역방향 이동 불가)
- **전체 시간복잡도**: O(N×d + P×d)

## 실패/버린 아이디어

- **구간별 직접 격자 탐색**: 각 구간에서 d차원 격자를 직접 탐색하는 방식
  → 경우의 수가 지수적으로 폭발 (C(N×d, N)^d 수준), 완전 불가능, 폐기
- **DP 접근**: 좌표별 DP 테이블 구성
  → d차원 상태 공간이 N^d 크기 → 메모리·시간 모두 불가능, 폐기

## 최종 선택 이유

- d차원 격자 경로를 "각 차원별 이동의 순열"로 보면, 이동 순서 결정 = 다항계수 공식 정확히 적용
- 구간 독립성: 중간점 통과 조건이 구간을 완전히 분리하므로 곱의 법칙 적용 가능
- 역팩토리얼 전처리로 나눗셈을 곱셈으로 대체 → O(1) per 다항계수 계산
- 구현 단순: 전처리 O(N×d) + 구간 루프 O(P×d) = 전체 O(N×d) 수준
